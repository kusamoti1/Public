# Public

底値メモ（PWA）

## 使い方（重要）
`https://github.com/kusamoti1/Public` はリポジトリ表示用URLです。アプリとして使うときは **GitHub Pages の公開URL**（通常 `https://kusamoti1.github.io/Public/`）を開いてください。

詳しい手順は `README.txt` を参照してください。

PRで競合（conflicts）が出る場合は、PRブランチが古い可能性があります。詳細は `README.txt` の「PRで「conflicts」が出るとき」を参照してください。
画面が古くてカメラ欄が出ない場合は、キャッシュが原因のことがあります。`README.txt` の「画面が古くて「カメラ欄」が出ないとき」を参照してください。
カテゴリ段階選択（例: 肉→豚→豚バラ）・バーコードガイド撮影・商品写真保存にも対応しました。
写真入力は「カメラで撮影」と「写真から選ぶ（端末保存画像）」の両方に対応しています。
